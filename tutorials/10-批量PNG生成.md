# 批量 PNG 生成 — 使用教程口播稿

---

## 开场

大家好，这一集我们来讲解批量 PNG 生成功能。

在科研工作中，我们经常需要把大量的衍射图像转换为 PNG 格式，用于论文插图、报告演示或者数据共享。手动一张一张导出非常耗时，批量 PNG 生成就是为了解决这个问题。

它支持从 H5、TIFF、EDF 文件夹批量生成色图映射的 PNG 图像，并且支持 JSON 配置文件，方便参数复用。

---

## 页面布局

页面分为左右两个区域：

- 左侧：文件夹设置、JSON 配置、显示设置、文件类型开关
- 右侧：扫描结果、运行按钮、进度、结果摘要、JSON 预览

---

## 第一步：设置源文件夹和输出文件夹

在左侧"文件夹"面板中：

- **源文件夹**：包含衍射图像文件的目录
- **输出文件夹**：PNG 文件的保存位置。选择源文件夹后，会自动设置为源文件夹下的 `export` 子目录

勾选"递归"可以包含子文件夹。

---

## 第二步：配置显示设置

### 色图
选择渲染 PNG 使用的色图。可选项包括 viridis、plasma、inferno、magma、cividis、jet、gray、hot、cool、turbo 等常用色图。

### 对数刻度
勾选后使用对数刻度渲染。对于强度动态范围大的数据，对数刻度可以同时显示强峰和弱信号。

### 对比度范围
- **自动**：软件自动计算最佳对比度范围
- **手动**：手动设置色标的最小值和最大值

### 输出布局
- **平铺**：所有文件的 PNG 直接输出到输出文件夹
- **按文件分组**：每个源文件创建一个子文件夹，对应的 PNG 放在子文件夹中

### 分辨率（DPI）
设置输出 PNG 的分辨率。默认 300 DPI，适合论文插图。如果需要更高分辨率，可以设置到 600 或更高。

### 显示选项
- **显示坐标轴**：在 PNG 图像上显示坐标轴刻度
- **显示色标**：在图像旁边显示颜色和强度的对应关系

---

## 第三步：设置文件类型

在"文件类型"面板中，勾选你要处理的文件格式：

- **TIFF / TIF**：通用图像格式
- **EDF / CBF**：同步辐射常用格式
- **H5 / HDF5**：层次数据格式

至少要勾选一种类型。

---

## 第四步：扫描源文件夹

点击"扫描"按钮，软件会扫描源文件夹中的所有匹配文件。

扫描完成后，会显示找到的文件列表，按类型分组：
- H5：找到的 HDF5 文件数量和列表
- TIFF：找到的 TIFF 文件数量和列表
- EDF：找到的 EDF 文件数量和列表

---

## 第五步：运行批量生成

确认设置无误后，点击"运行批量生成"按钮。软件会逐个处理文件，进度条显示当前进度。

完成后会显示结果摘要：
- 总数：扫描到的文件总数
- 成功：成功生成的 PNG 数量
- 失败：失败的数量
- 耗时：总处理时间

---

## JSON 配置功能

批量 PNG 生成支持 JSON 配置文件，这是一个非常实用的功能。

### 保存配置
点击"保存 JSON"按钮，可以把当前的所有设置保存为一个 JSON 文件。这个文件包含了色图、对比度、DPI 等所有参数。

### 加载配置
点击"加载 JSON"按钮，选择一个之前保存的 JSON 文件，所有设置会自动恢复。

### JSON 预览
页面底部会显示当前配置的 JSON 内容，方便你检查和复制。

这个功能的价值在于：
- **参数复用**：保存一套常用的配置，下次直接加载
- **团队共享**：把配置文件分享给同事，确保大家使用相同的渲染参数
- **可重复性**：论文审稿人可以用相同的配置复现你的图像

---

## 实际应用场景

1. **论文准备**：把所有衍射图像批量转换为统一风格的 PNG
2. **数据共享**：把 HDF5 数据转换为更容易查看的 PNG 格式
3. **报告制作**：批量生成用于学术报告的插图
4. **质量检查**：快速浏览大量数据的整体质量

---

## 注意事项

1. 输出文件夹如果不存在会自动创建
2. 对于非常大的文件，PNG 生成可能需要较长时间
3. 高 DPI 设置会显著增加文件大小和处理时间
4. H5 文件中的多帧数据会每帧生成一个 PNG
5. 保存的 JSON 配置文件可以在不同机器之间共享使用
6. 如果源文件夹中的文件很多，扫描过程可能需要一些时间
