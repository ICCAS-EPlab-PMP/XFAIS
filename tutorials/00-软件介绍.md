## 开场

大家好，欢迎来到 X-FAIS 的使用教程系列视频。

X-FAIS，全称是 X-ray pyFAI Scattering & Diffraction Suite，是一款专为 X 射线衍射数据处理设计的桌面软件。它基于 pyFAI 和 fabio 这两个成熟的科学计算库，把原本需要写 Python 脚本才能完成的积分、转换、导出等操作，全部封装成了图形化界面，让大家点点鼠标就能完成工作。

---

## 软件定位

X-FAIS 主要面向以下场景：

- **WAXS / SAXS**：广角和小角 X 射线散射数据的 1D、2D 积分
- **GIWAXS**：掠入射广角 X 射线散射的纤维衍射积分
- **图像查看**：EDF、TIFF、HDF5 等格式的探测器图像快速浏览
- **数据转换**：HDF5 文件的格式转换和数据提取
- **标样生成**：从 CIF 文件或晶胞参数生成理论衍射峰位

---

## 技术架构

简单介绍一下软件的架构，帮助大家理解它的运行方式：

1. **前端界面**：基于 Vue 3 构建，所有操作都在一个桌面窗口里完成
2. **后端计算**：内嵌了一个 Python 3.11.9 运行环境，通过 WebSocket 和前端通信
3. **核心引擎**：pyFAI 负责积分计算，fabio 负责图像文件读写，h5py 负责 HDF5 数据访问

也就是说，你不需要自己安装 Python 或者任何依赖，软件开箱即用。

## 首页导航

打开软件后，你会看到首页分为四个功能区域：

### 第一行：常用积分
- **1D 径向积分**：最常用的功能，将 2D 衍射图转换为强度随 q 或 2θ 变化的曲线
- **方位角 χ 积分**：分析强度随方位角的分布
- **CAKE 扇区积分**：只对 Debye-Scherrer 环的某个扇区做径向积分

### 第二行：掠入射衍射
- **GIWAXS 纤维积分**：生成 q_ip × q_oop 的二维强度图

### 第三行：图像工具
- **图像查看器**：浏览衍射图像，支持帧导航、对比度调节、PNG 导出
- **H5 格式转换**：将 HDF5 文件批量转换为 TIFF 或 CSV
- **H5 数据提取**：从 HDF5 文件中提取指定数据集

### 第四行：PyFAI 辅助功能
- **标样生成器**：从 CIF 文件或晶胞参数计算理论衍射峰位
- **pyFAI 标定检查**：检查系统中 pyFAI_calib2 的安装状态

## 通用操作说明

在后续的教程中，有几个通用概念会反复出现，这里先统一说明：

### 几何参数（Geometry）
积分需要知道探测器的几何关系，包括：
- **像素大小**（pixel1、pixel2）：探测器单个像素的物理尺寸，单位是米
- **样品-探测器距离**：单位是米
- **X 射线波长**：单位是埃（Å）
- **光束中心**（centerX、centerY）：直射光束在探测器上的像素坐标
- 也可以直接加载 PONI 文件，一次性导入所有几何参数

### 掩膜（Mask）
用于排除探测器上的坏像素、光束挡板等区域。支持：
- 值范围掩膜：排除强度低于或高于阈值的像素
- 死像素检测：自动标记异常像素
- 自定义掩膜文件：加载外部掩膜图像

### 导入模式
所有支持文件选择的页面都有两种导入模式：
- **替换模式**：选择新文件时清空之前的列表
- **追加模式**：将新文件添加到已有列表中

### 导出功能
积分完成后，支持将结果导出为多种格式：
- TXT / CSV：纯文本表格
- HDF5：结构化数据
- XY：两列数据格式
